The method

A coaching model built on seeing, not telling.

Most PD describes good teaching. We show it — then make it repeatable. Three steps, one tight loop.

  1. 01

    See it

    Watch a short, real exemplar clip of the exact skill in action — so the standard is concrete, not abstract.

  2. 02

    Name it

    Pinpoint the single highest-leverage move and name the gap in plain, teacher-friendly language.

  3. 03

    Do it

    Practice the move, get real-time feedback, and embed it in the very next lesson. Then repeat.
